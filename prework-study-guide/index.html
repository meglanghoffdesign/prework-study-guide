HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for HyperText Markup Language which is the standard language used to create and structure content on the web.
  • HTML uses tags to define elements. Tags are enclosed in angle brackets (< >). HTML elements typically have an opening tag and a closing tag with content placed in between.
  • Every HTML document should include key elements such as !DOCTYPE, html, head, and body
  • HTML headings range from h1 to h6. The h1 tag is for the main heading and h6 is for the least important heading. These help define the hierarchy of content.
  • HTML elements can have attributes. Attributes provide additional information about elements and are included in the opening tag.

Git

  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switch to main branch
  • git pull origin main: pulls down latest main branch
