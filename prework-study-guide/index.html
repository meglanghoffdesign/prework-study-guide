HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for HyperText Markup Language which is the standard language used to create and structure content on the web.
  • HTML uses tags to define elements. Tags are enclosed in angle brackets (< >). HTML elements typically have an opening tag and a closing tag with content placed in between.
  • Every HTML document should include key elements such as !DOCTYPE, html, head, and body.
  • HTML headings range from h1 to h6. The h1 tag is for the main heading and h6 is for the least important heading. These help define the hierarchy of content.
  • HTML elements can have attributes. Attributes provide additional information about elements and are included in the opening tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • It best to avoid inline styles and keep the CSS in a separate stylesheet for easier maintenance.
  • Keep it DRY: Don't Repeat Yourself—reuse styles where possible.
  • Utilize dev tools by using browser dev tools to inspect and debug your CSS

Git

How to Start

  • cd DIRECTORY NAME to navigate to the correct directory.
  • git status to check git branch (should be on main).
  • Not on the main branch? git checkout main to switch to main.
  • git pull origin main to pull down latest main branch to local.
  • git checkout -b BRANCH NAME(ex. feature/add-html) to create new feature branch.
  • Adding CSS? touch style.css into assets directory.
  • Adding Javascript? touch script.js into assets directory.
  • code . to open VS Code.


How to Save

  • cd DIRECTORY NAME to navigate to the correct directory.
  • git add -A to push all content to the next commit.
  • git commit -m “< MESSAGE >” to add context to commit/commits locally. (no space around message)
  • git pull origin main to pull down latest main branch to local.
  • git push origin BRANCH NAME(ex. feature/add-html) to push changes to remote repository.

JavaScript

  • A variable is a container used to store data that can be used or updated later.
  • Semicolons (;) are used to separate statements in JavaScript.
  • const declares a constant variable whose value cannot be changed after it's assigned.
  • A string is a sequence of characters (letters, numbers, symbols) surrounded by quotation marks.
  • console.log(variable) prints the value of variable to the browser’s console (useful for debugging)
  • Control flow is the order in which code is executed based on conditions or loops.
  • An if statement prints a block of code if a condition is true.
  • An else if statement provides another condition to check if the first if condition is false.
  • = is used for assignment while === checks strict equality, meaning both value and type must be the same
  • An array is a list of items (values) stored in a single variable.
  • The index is the position of an element in an array, ALWAYS starting at 0.
  • A for loop is used to repeat a block of code a specific number of times.
  • While const declares a constant variable that cannot be changed, let declares a variable whose value can be updated.
  • A Boolean is a data type that can only be true or false.
  • x++ is short for x + 1
  • For loops have three parts: The for keyword. A three-part conditional statement: (let x = 0), condition (x < 5), and update (x++). A code block to execute on each loop iteration.
